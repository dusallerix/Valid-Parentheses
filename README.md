# Valid-Parentheses
Задача с Leetcode
## Описание задачи


Дана строка s, содержащая только символы '(', ')', '{', '}', '[' и ']'. Определите, является ли входная строка допустимой.  
  
Входная строка считается допустимой, если:
- Открывающие скобки должны закрываться скобками того же типа.  
- Открывающие скобки должны закрываться в правильном порядке.
- Каждой закрывающей скобке соответствует открывающая скобка того же типа.
    
Example 1:    
>Input: s = "()"  
>Output: true  
    
Example 2:    
>Input: s = "()[]{}"  
>Output: true  
    
Example 3:    
>Input: s = "(]"    
>Output: false    
    
Example 4:    
>Input: s = "([])"    
>Output: true    
    
Example 5:    
>Input: s = "([)]"  
>Output: false  
## Логика решения
Исходя из условия и примеров, а также на основе Примера 4 был сделан такой алгоритм:
1) Заводим пустой массив, в который будем добавлять наши символы через цикл.
2) Открывающие "скобки" (так я называю символы "(", "[" и "{") мы без проверок добавляем в наш массив.
3) В случае закрывающих скобок, мы делаем проверку, исходя из которой сравниваем последний символ в массиве (т.е. предыдущий). Если предыдущий символ является скобкой того же типа что и закрывающая, то он удаляется из массива, а текущий символ не добавляется в массив.
4) Итог, после цикла делаем return [...] == 0. В конце цикла, если все условия были соблюдены, то массив не будет иметь в себе ни единого элемента и соответственно в результате этого сравнения вернет true.
